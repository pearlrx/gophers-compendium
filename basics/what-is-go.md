# What Is Go?

Go (Golang) is an open-source, compiled, statically typed programming language created at Google.  
It focuses on simplicity, performance, and easy concurrency.

## Key Features
- Clean and minimal syntax
- Fast compilation
- Built-in concurrency (goroutines, channels)
- Strong standard library
- Great developer tools (`go build`, `go test`, `go fmt`)

## Common Use Cases
- Backend services
- Cloud and DevOps tools
- CLI applications
- Networking and distributed systems
